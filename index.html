CSS Cheat Sheet

Multiple Shadows

Use this code to style multiple shadows on an h1 header. The values can be manipulated to achieve different effects. 

h1 {
    text-shadow:
        1px 1px 1px red,
        2px 2px 1px red;
}

Media Queries with and logic

Use the and operator to combine a media type and feature.

@media screen and (min-width: 600px) and (orientation: landscape) {
body {
    color: blue;
    }
}

Pseudo Classes

Use pseudo classes to manipulate elements in a specific state. In this case, the first child p element in an article element. 

article p:first-child {
    font-size: 120%;
    font-weight: bold;
}

Declare custom properties

Declare custom properties in :root and reference them throughout your stylesheet! These are the colors used on this page!

:root {
    --primary-gray: #484444;
    --secondary-gray: #AAAAAA;
}

Reference custom properties

Once declared in root, call reference to your variable.

.class {
    color: var(--primary-gray);
}

Set a gradient background

Use this code to set a linear gradient background.

html {
    background: linear-gradient(#484444, #AAAAAA);
}
